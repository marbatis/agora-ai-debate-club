# Screenshots Instructions

## Required Screenshots for Kaggle Submission

Please add the following screenshots to this directory:

### 1. web-interface.png
- Screenshot of the `demo_client.html` web interface
- Should show the input form with a debate topic entered
- Recommended size: 1200x800px

### 2. debate-results.png
- Screenshot showing a completed debate with results
- Should display:
  - Both debaters' arguments
  - Judge scoring breakdown
  - Winner announcement
- Recommended size: 1200x1000px

### 3. architecture-diagram.png
- Visual representation of the multi-agent system
- Can be created using:
  - Excalidraw (https://excalidraw.com/)
  - Draw.io (https://draw.io/)
  - Mermaid rendered to PNG
- Should show: Debater A, Debater B, Judge, and their interactions

## How to Take Screenshots:

### On Mac:
- Cmd + Shift + 4 - Select area screenshot
- Cmd + Shift + 3 - Full screen screenshot

### On Windows:
- Windows + Shift + S - Snipping tool
- Or use Snip & Sketch

## To Create Architecture Diagram:

1. Visit https://excalidraw.com/
2. Draw the architecture showing:
   - User Input → Debate Manager
   - Debate Manager → Debater A & B
   - Debaters → Judge
   - Judge → Results
3. Export as PNG
4. Save as `architecture-diagram.png`
